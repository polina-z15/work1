  • git diff- Перераховує всі нові або змінені файли, які потребують фіксації
  • git add [файл]- Показує відмінності щодо внесених змін у ще не проіндексованих файлах
  • git reset [файл]- Показує різницю між проіндексованою та останньою зафіксованою версіями файлів
  • git diff --staged- Індексує вказаний файл для наступного коміту
